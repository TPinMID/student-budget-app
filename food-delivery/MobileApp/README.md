# MobileApp — мобильное приложение

## Что находится в этой папке
Код мобильного приложения для клиента сервиса доставки еды. Основной сценарий: выбрать ресторан → собрать корзину → оплатить → отслеживать заказ → получить доставку.

## Планируемые компоненты/модули
- **Auth**: регистрация, вход, восстановление доступа
- **Catalog**: список ресторанов, категории, блюда, карточка блюда
- **Cart & Checkout**: корзина, адрес, комментарии к заказу, подтверждение
- **Payment**: интеграция оплаты (провайдер уточняется), статусы платежей
- **Orders**: история заказов, детали заказа, повтор заказа
- **Tracking**: статусы (готовится/в пути/доставлено), карта и трекинг курьера (если включено)
- **Profile**: адреса, настройки, поддержка
- **Notifications**: push-уведомления о статусах заказа и промо (опционально)

## Взаимодействие с другими частями проекта
- Приложение обращается к **ServerApi** для:
  - авторизации и получения токенов;
  - получения списка ресторанов/меню;
  - создания и оплаты заказа;
  - получения статусов заказа и трекинга.
- Для отображения контента (баннеры, условия, справка) может использовать данные/страницы из **Web** (если решим не дублировать контент в приложении).
